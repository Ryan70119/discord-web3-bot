# Discord Web3 Bot

A simple Discord bot that returns cryptocurrency prices.

Built using **discord.js** and the CoinGecko API.

## Features

* Responds to `!price` command
* Fetches Bitcoin and Ethereum prices
* Simple JavaScript Web3 utility bot

## Installation

Install dependencies:

npm install

## Run the bot

node bot.js

## Example command

!price

Bot response:

BTC: $67000
ETH: $3500
